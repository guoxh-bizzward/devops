# 不借助第三变量交换两个变量值的方案

* 先回顾下利用第三变量的实现方式
* 终极版本一:单纯数学运算方式

```
let a = 5;
let b = 4;

a = a + b;
b = a - b;
a = a - b;
```

* 终极版本二:位运算符 `^`异或

异或是执行位运算的,二进制运算.参与变量运算的两个变量要转为二进制计算.如果相同二进制的值相同,则转为0,否则为1

```
let a = 5; 0101
let b = 4; 0100

a = a ^ b; //0101^0100 = 0001 ,此时 a=1
b = a ^ b; //0001 ^ 0100 = 0101,此时 b=5;
a = a ^ b; //0001 ^ 0101 = 0100, 此时 a=4;
```
